Cài đặt môi trường C++

Hướng dẫn chi tiết cài đặt Code::Blocks phiên bản 25.03 kèm trình biên dịch MinGW.

1 Tải về (Quan trọng nhất)

Hầu hết lỗi xảy ra do tải sai phiên bản. Bạn cần phiên bản có kèm Compiler (Trình biên dịch).

Chú ý tuyệt đối:

Phải tải file có tên: codeblocks-25.03mingw-setup.exe
(Có chữ "mingw" là đúng)

Tải Code::Blocks 25.03 (Kèm MinGW) Link an toàn từ Google Drive / SourceForge

2 Các bước cài đặt

3 Kích hoạt Compiler

Trong lần khởi động đầu tiên, bảng Compiler auto-detection sẽ hiện ra.

1. Tìm dòng: GNU GCC Compiler.

2. Nhấn vào nó.

3. Nhấn nút Set as default bên phải.

4. Nhấn OK.

4 Viết Code đầu tiên

main.cpp 
#include <iostream>

using namespace std;

int main() {
    cout << "Hello world!" << endl;
    return 0;
}
Cách tạo dự án:
  • File -> New -> Project
  • Chọn Console Application
  • Chọn C++
  • Đặt tên (VD: HelloWorld)

5 Chạy chương trình

Nhấn phím tắt thần thánh:

F9

(Hoặc tìm nút Build and Run hình bánh răng + tam giác xanh)

> Hello world!

Process returned 0 (0x0) execution time : 0.051 s

Press any key to continue.

Gặp lỗi "Can't find compiler executable"?

Lỗi này 99% là do bạn tải bản không có MinGW.
Giải pháp: Gỡ cài đặt Code::Blocks hiện tại, quay lại Bước 1 và tải đúng file được chỉ định.
